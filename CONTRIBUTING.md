Вивчіть проект: Зрозумійте його цілі, структуру та правила (README, CONTRIBUTING.md).
Створіть гілку: Працюйте в окремій гілці для вашого завдання.
Робіть маленькі комміти: Кожен комміт має вирішувати одне завдання.
Пишіть зрозумілий код/текст: Дотримуйтесь стилю проекту, коментуйте.
Напишіть тести: Переконайтеся, що код працює коректно.
Створіть Pull Request (PR): Опишіть зміни та попросіть ревью.
Будьте готові до реву: Вносите правки за зауваженнями, будьте ввічливі.
